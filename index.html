Point 1 : How to install the extension ?

First of all, to use our extension you will need to install it.
You can find it in the Chrome Web Store by clicking here : Click !
Once there, simply click on "Add to Chrome" as shown in the image below

You will be asked if you really want to add the extension, click on "add extension"
Congratulations ! You installed the Ekami Extension you were looking for !

Point 2 : How does the extension works ?

The extension will automatically be displayed with the Ekami logo in the extension bar of your browser. Here's an example in Chrome :

Now to use it you can click on it. You are now using our extension ! But if you just do this you will be quite limited.
As you can see, once you click on the extension logo, a little popup opens. It will be the main tool to choose how you want to use it.

The text input

The three icons at the top are not usefull for your work, they display some informations if needed. What's interesting is the text input below, with the red trash icon and the blue arrow icon.
If you have a barcode scanner you can click on the text input to give it the focus, then scan your QR Code with your scanner. It will type in the input and enter, if the QR Code is correct it will fill the form !

The position Settings and Scan Settings

This part is usefull if you want to use our inline Scanner. To do so you can select if you want to use your camera, a simple text input or even both !
You can also choose the position in the webpage, for example if you check the "Bottom Right" radio button, the inline Scanner will go to the bottom right of the webpage it's on.
And since you choosed how you wanted to use the inline scanner on the webpage, every time this specific page loads you will see the inline Scanner with your settings.
You can remove the inline Scanner by unchecking both "Camera Scan" and "Manual Scan" radio buttons.

The Webcam Selector dropdown menu

This little dropdown menu will display every video device (webcams) connected to your system. You can choose wich one you want to use, it will be remembered as a setting as mentionned above
